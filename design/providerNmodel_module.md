# 模型供应商与模型模块

## 需求概述
   需要是一个模型供应商与模型管理模块，用于接入和管理当前市面上大多数模型供应商并进行模型配置。
   该模块需要基于llmlite的功能进行拓展。
   需要实现的功能如下：
   1.可以创建自定义添加或是选择预设已有的模型供应模板，来创建模型。
   2.在保存时候，提供一个测试接口，来测试是否能从当前模型实例去访问回答。
   3.可以选择配置的模型是否支持图片生成，视觉功能，思考功能能特性。
   4.在界面上可以根据已有的供应商类别来进行筛选
   5.可对模型名称进行模糊查询
   6.管理员可以配置该模型是否可用
   7.在创建模型实例之前，需要提前维护一个供应商的信息，然后将模型实例绑定至供应商上。
   8.模型实例创建的时候，可以通过一个字典文本，去配置模型的temperature，top等一系列参数，进行微调，以llmlite中的配置为准。
   9.需要注意该模块创建的所有模型，都会通过llmlite-proxy，使用外提供调用。
   10.每个供应商下可配置多个api-key，在配置模型实例的时候，可以点选对应的可用的api-key进行配置。
