# api调用与授权模块

## 需求概述
    在该平台所配置的所有可用的模型实例，可以通过被外部服务使用。
    该部分的供给外部调用特性需要基于llmlite-proxy实现。
    1.管理员可以使用一个API授权清单，将可用的模型实例，mcp和skill等功能和用户组或用户绑定。
    2.用户或用户组中的用户该授权清单来获取模型的调用接口和TOKEN进行调用。
    3.需要将外部调用API接口的操作日志，写入审计日志。
